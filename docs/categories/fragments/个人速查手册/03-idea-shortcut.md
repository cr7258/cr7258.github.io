## Jetbrains 快捷键

|  快捷键   | 功能  |
|  ----  | ----  |
| **Command** + **Shift** + **+**  | 展开所有代码 |
| **Command** + **Shift** + **-**  | 折叠所有代码 |

## VS Code 快捷键

|  快捷键   | 功能  |
|  ----  | ----  |
| **Shift** + **Option** + **F**  | 格式化代码 |
| **Command** + **D**  |  选择下一个相同的文本，可以同时编辑多个相同的文本 |
